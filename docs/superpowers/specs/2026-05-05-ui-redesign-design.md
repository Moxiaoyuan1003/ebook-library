# UI 现代化重设计 + 新功能设计文档

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将个人图书管理系统从基础暗色界面升级为三套主题可切换的现代化 UI，同时新增阅读统计、书籍详情、全文搜索等功能页面，并补充元数据获取、封面提取、拖拽导入、笔记导出等实用功能。

**Architecture:** 基于 zustand themeStore 扩展为完整主题 token 系统，三套主题（毛玻璃/简洁/炫彩）通过 CSS 变量 + React context 下发。新增页面复用现有路由结构和 API 层。

**Tech Stack:** React + TypeScript + Ant Design + zustand + react-router-dom + recharts（统计图表）+ react-markdown（AI 聊天渲染）

## 1. 主题系统

### 1.1 设计目标

三套视觉主题可一键切换，覆盖应用所有页面和组件。主题偏好持久化到 localStorage。

### 1.2 三套主题 Token 定义

#### 毛玻璃主题 (Glass)

```
bgGradient: ['#0f0c29', '#302b63', '#24243e']
sidebar:    rgba(255,255,255,0.04) + backdrop-filter: blur(12px)
header:     rgba(255,255,255,0.06) + backdrop-filter: blur(12px)
card:       rgba(255,255,255,0.08) + backdrop-filter: blur(12px) + border: 1px solid rgba(255,255,255,0.12)
primary:    linear-gradient(135deg, #667eea, #764ba2)
text:       #eee
textSecondary: rgba(255,255,255,0.5)
border:     rgba(255,255,255,0.06)
radius:     16px
shadow:     0 8px 32px rgba(0,0,0,0.2)
```

#### 简洁主题 (Clean)

```
bg:         #0f1117
sidebar:    #1a1d27
header:     #161822
card:       #1a1d27 + border: 1px solid #2a2d37
primary:    #3b82f6
text:       #e5e7eb
textSecondary: #6b7280
border:     #2a2d37
radius:     12px
shadow:     none (用 border 代替)
```

#### 炫彩主题 (Vibrant)

```
bg:         #0a0a0a
sidebar:    #111111
header:     #0d0d0d
card:       #111111 + glow effect
primary:    linear-gradient(135deg, #8b5cf6, #ec4899)
text:       #eeeeee
textSecondary: #666666
border:     #1a1a1a
radius:     16px
shadow:     0 4px 16px rgba(139,92,246,0.15)
glowAccent: rgba(139,92,246,0.3)
```

### 1.3 主题 Token 结构

```typescript
interface ThemeTokens {
  name: string;
  // 背景
  bg: string;
  bgGradient?: string[];
  // 布局区域
  sidebar: string;
  header: string;
  content: string;
  // 卡片
  cardBg: string;
  cardBorder: string;
  cardShadow: string;
  // 主色
  primary: string;
  primaryGradient?: string[];
  // 文字
  text: string;
  textSecondary: string;
  textMuted: string;
  // 边框 & 圆角
  border: string;
  radius: number;
  // 特效
  blur?: number;
  glow?: string;
  // 阅读器专用
  readerBg: string;
  readerText: string;
}
```

### 1.4 themeStore 扩展

扩展现有 `frontend/src/stores/themeStore.ts`：

- `themeName: 'glass' | 'clean' | 'vibrant'`
- `tokens: ThemeTokens`（当前主题的完整 token）
- `setTheme(name)` — 切换主题
- `readerBgMode: 'theme' | 'eyecare-warm' | 'eyecare-green'` — 阅读器背景模式
- `setReaderBgMode(mode)` — 切换阅读器背景
- 保留现有 `bgColor/sidebarColor/headerColor/contentColor` 的自定义覆盖能力

### 1.5 主题应用方式

- 所有组件通过 `useThemeStore(s => s.tokens)` 获取当前主题 token
- 组件内使用 token 值替代硬编码颜色
- CSS 变量同步注入 `:root`，方便第三方组件（如 Ant Design）覆盖

### 1.6 设置页主题切换

- 三套主题的实时预览卡片（小尺寸 App 截图）
- 点击即切换，立即全局生效
- 护眼色选项独立于主题，仅影响阅读器背景

## 11. 附加功能（按优先级）

### P0 — 必须有（与核心改造同步交付）

#### 11.1 拖拽导入

- 支持将文件直接拖拽到书库页面任意区域
- 拖拽时显示全屏蓝色遮罩 + "释放以导入" 提示
- 支持多文件同时拖入
- 复用现有 `importFile` API

#### 11.2 封面提取

- PDF：使用 `pdfjs-dist` 渲染第一页为缩略图
- EPUB：解析 OPF 获取 cover image
- 提取的封面存储到 `backend/covers/` 目录
- 书籍卡片和详情页优先显示真实封面，无封面时回退到渐变色块
- 后端新增 `GET /api/books/:id/cover` 返回封面图片

#### 11.3 ISBN 元数据自动获取

- 导入书籍时，如果有 ISBN，自动查询 Open Library API (`https://openlibrary.org/isbn/{isbn}.json`)
- 自动填充：书名、作者、出版年、封面图 URL
- 查询失败时静默跳过，不阻塞导入
- 详情页增加 "刷新元数据" 按钮，手动触发

#### 11.4 全局快捷键

| 快捷键 | 功能 |
|--------|------|
| `/` | 聚焦搜索栏（跳转 /search） |
| `Ctrl+N` | 打开导入对话框 |
| `Ctrl+,` | 打开设置 |
| `Ctrl+Shift+S` | 打开阅读统计 |
| `Escape` | 关闭当前弹窗/面板 |

- 阅读器内的快捷键保持不变（A/D/F 等）
- 快捷键在 textarea/input 中不生效
